# 🚀 AI Spam Detection App

A simple end-to-end Machine Learning project that detects whether a message is **Spam** or **Not Spam**, with a deployed web interface.

---

## 📌 Project Overview

This project demonstrates the complete AI pipeline:

- Data preprocessing
- Model training using Machine Learning
- Model saving and loading
- Deployment using Streamlit
- User-friendly GUI interface

---

## 🧠 Model Details

- Algorithm: Logistic Regression
- Text Processing: TF-IDF Vectorization
- Library: scikit-learn
